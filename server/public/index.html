Bank Marketing API

Welcome to the Bank Marketing API. This API provides access to various endpoints related to bank marketing data.

  • /api/v1/customers - Get all customers
  • /api/v1/customers/:id - Get customer by ID
  • /api/v1/bank-marketing - Get all marketing campaigns
  • /api/v1/bank-marketing/:id - Get marketing campaign by ID
  • /api/v1/bank-marketing/create - Create a new marketing campaign
  • /api/v1/bank-marketing/update/:id - Update a marketing campaign by ID
  • /api/v1/bank-marketing/delete/:id - Delete a marketing campaign by ID
